# Teka

Data storage project

(This project is under development.)

## Demo

[teka-storage](https://teka-storage.netlify.app/)

## Project Setup

1. 
```sh
git clone https://github.com/AL5t/Teka.git
```

2. 
```sh
cd teka
```

3. 
```sh
npm install
```

## Technology Stack

- Vue 3
- Vite
- Pinia
- Primevue
- IndexedDB
